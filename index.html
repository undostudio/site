  • Flowstash (coming soon) — I've always found plain text files to be the best way to organize small projects. Even for small teams, a Google Doc style task list is wonderful compared to having to navigate the UI of any issue tracking tool. It's flexible and robust. The number of clicks required to create a GitLab issue is embarrassing. Flowstash is an editor that syncs with popular issue trackers to display issues as text. Edit the issues in Flowstash, and they're automatically updated in the issue tracking tool. Create a new issue? Just make a new paragraph.
